# FinTrack – Personal Finance Tracker

A mobile-first personal finance app built with Flutter & Firebase.

🌐 **Web:** https://fintrack-a459c.web.app

## Features

- Track checking accounts, fixed deposits, ETFs, crypto, physical assets & debts
- Net worth history with interactive chart
- Live prices for crypto, ETFs & commodities (Binance, Yahoo Finance, CoinGecko)
- Upcoming Festgeld maturities — stacked notifications on home screen, collapsible
- Push notifications for fixed deposit maturity dates (30, 7, 1 day before & on due date)
- Calendar integration — add due dates directly to your calendar app
- Dark & Light Mode
- Google Sign-In
